About Reel Match

We created a dynamic web page that generates movies via a search bar and displays movie information. We also have a functioning “favorite” button in order to like movies and save them to your profile. If you aren’t sure about the movie try watching the movie trailer by clicking the trailer button! Most importantly this viewable profile displays user favorite movies, and then generates a list of movie recommendations.
